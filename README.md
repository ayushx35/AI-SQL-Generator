# 🤖 AI-SQL-Generator

AI-SQL-Generator is a Streamlit-based web application that allows you to interact with your SQL database using natural language.  
It converts plain English queries into SQL statements using OpenAI and executes them on your connected database.

---

## 🚀 Features

- 🧠 Natural Language to SQL conversion
- 📊 Supports PostgreSQL, MySQL, and SQLite
- 🌐 Interactive Streamlit UI
- 🔑 OpenAI GPT-powered query generation
- ⚡ Automatic SQL execution and response display

---

## 🏗️ Tech Stack

- Python 3.9+
- Streamlit
- LangChain
- SQLAlchemy
- OpenAI API
- PostgreSQL / MySQL / SQLite

---

## 📦 Installation

### 1 Clone the Repository

```bash
git clone https://github.com/ayushx35/AI-SQL-Generator.git
cd AI-SQL-Generator

```

### 2 Install Dependencies

```bash
pip install -r requirements.txt
```

### 3 Running the Application

```bash
streamlit run app.py
```
After running, open the URL shown in the terminal (usually):
```bash
http://localhost:8501
```
